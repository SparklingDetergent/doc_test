**大規模言語モデル**（だいきぼげんごモデル、[英](/wiki/%E8%8B%B1%E8%AA%9E "英語")
: large language model、**LLM**）は、多数のパラメータ（数千万から数十億）を持つ[人工ニューラルネットワーク](/wiki/%E3%83%8B%E3%83%A5%E3%83%BC%E3%83%A9%E3%83%AB%E3%83%8D%E3%83%83%E3%83%88%E3%83%AF%E3%83%BC%E3%82%AF "ニューラルネットワーク")
で構成されるコンピュータ[言語モデル](/wiki/%E8%A8%80%E8%AA%9E%E3%83%A2%E3%83%87%E3%83%AB "言語モデル")
で、膨大なラベルなしテキストを使用して[自己教師あり学習](/wiki/%E8%87%AA%E5%B7%B1%E6%95%99%E5%B8%AB%E3%81%82%E3%82%8A%E5%AD%A6%E7%BF%92 "自己教師あり学習")
または[半教師あり学習](/w/index.php?title=%E5%8D%8A%E6%95%99%E5%B8%AB%E3%81%82%E3%82%8A%E5%AD%A6%E7%BF%92&action=edit&redlink=1 "「半教師あり学習」 (存在しないページ)")
（[英語版](https://en.wikipedia.org/wiki/Semi-supervised_learning "en:Semi-supervised learning")
）によって訓練が行われる[\[1\]](#cite_note-:1-1)
。

LLMは2018年頃に登場し、さまざまなタスク（仕事）で優れた性能を発揮している。これにより、[自然言語処理](/wiki/%E8%87%AA%E7%84%B6%E8%A8%80%E8%AA%9E%E5%87%A6%E7%90%86 "自然言語処理")
の研究の焦点は、特定のタスクに特化した[教師あり](/wiki/%E6%95%99%E5%B8%AB%E3%81%82%E3%82%8A%E5%AD%A6%E7%BF%92 "教師あり学習")
モデルを訓練するという以前の[パラダイム](/wiki/%E3%83%91%E3%83%A9%E3%83%80%E3%82%A4%E3%83%A0 "パラダイム")
から転換した[\[2\]](#cite_note-Manning-2022-2)
。大規模言語モデルの応用は目覚ましい成果を上げているが、大規模言語モデルの開発はまだ始まったばかりであり、多くの研究者が大規模言語モデルの改良に貢献している[\[3\]](#cite_note-3)
。

大規模言語モデルという用語の正式な定義はないが、大規模[コーパス](/wiki/%E3%83%86%E3%82%AD%E3%82%B9%E3%83%88%E3%82%B3%E3%83%BC%E3%83%91%E3%82%B9 "テキストコーパス")
で事前訓練された、数百万から数十億以上のパラメータを持つ[ディープラーニング](/wiki/%E3%83%87%E3%82%A3%E3%83%BC%E3%83%97%E3%83%A9%E3%83%BC%E3%83%8B%E3%83%B3%E3%82%B0 "ディープラーニング")
モデルを指すことが多い。LLMは、特定のタスク（[感情分析](/wiki/%E6%84%9F%E6%83%85%E5%88%86%E6%9E%90 "感情分析")
、[固有表現抽出](/wiki/%E5%9B%BA%E6%9C%89%E8%A1%A8%E7%8F%BE%E6%8A%BD%E5%87%BA "固有表現抽出")
、[数学的推論](/wiki/%E8%87%AA%E5%8B%95%E6%8E%A8%E8%AB%96 "自動推論")
など）のために訓練されたものとは異なり、幅広いタスクに優れた汎用モデルである[\[2\]](#cite_note-Manning-2022-2)
[\[4\]](#cite_note-emergentpaper-4)
。LLMがタスクを実行する能力や対応可能な範囲は、ある意味では設計における画期的な進歩には依存せず、LLMに費やされた資源（データ、パラメータサイズ、計算力）の量の関数であるように見える[\[5\]](#cite_note-Bowman-5)
。多数のパラメータを持ったニューラル言語モデルは、文の次の単語を予測するという単純なタスクで十分に訓練することで、人間の言葉の構文や意味の多くを捉えられることがわかった。さらに、大規模な言語モデルは、世の中に関するかなりの一般知識を示し、訓練中に大量の事実を「記憶」することができる[\[2\]](#cite_note-Manning-2022-2)
。

質の高い証拠とされる2023年のメタ分析によれば、大規模言語モデルの創造性に目を輝かせる研究者はもちろん世界中に存在し、小規模言語モデルにはできないタスクで大規模言語モデルが創造的であると主張する学者もいるが、これは測定基準の選択によるものであり、創造性によるものではないことが示唆されている。異なる測定基準を選択した場合、大規模言語モデルの創造性の優位性は見られない可能性が示唆されている[\[6\]](#cite_note-6)
。

### 事前訓練データセット\[[編集](/w/index.php?title=%E5%A4%A7%E8%A6%8F%E6%A8%A1%E8%A8%80%E8%AA%9E%E3%83%A2%E3%83%87%E3%83%AB&action=edit&section=2 "節を編集: 事前訓練データセット")\
\]

「[機械学習研究のためのデータセットリスト](/w/index.php?title=%E6%A9%9F%E6%A2%B0%E5%AD%A6%E7%BF%92%E7%A0%94%E7%A9%B6%E3%81%AE%E3%81%9F%E3%82%81%E3%81%AE%E3%83%87%E3%83%BC%E3%82%BF%E3%82%BB%E3%83%83%E3%83%88%E3%83%AA%E3%82%B9%E3%83%88&action=edit&redlink=1 "「機械学習研究のためのデータセットリスト」 (存在しないページ)")
（[英語版](https://en.wikipedia.org/wiki/List_of_datasets_for_machine-learning_research "en:List of datasets for machine-learning research")
）」も参照

大規模言語モデル（LLM）は通常、さまざまな分野や言語にわたる大量のテキストデータで事前訓練が行われる[\[7\]](#cite_note-7)
。著名な事前訓練データとしては、[Common Crawl](/wiki/Common_Crawl "Common Crawl")
、[The Pile](/w/index.php?title=The_Pile_(%E3%83%87%E3%83%BC%E3%82%BF%E3%82%BB%E3%83%83%E3%83%88)&action=edit&redlink=1 "「The Pile (データセット)」 (存在しないページ)")
（[英語版](https://en.wikipedia.org/wiki/The_Pile_(dataset) "en:The Pile (dataset)")
）、MassiveText[\[8\]](#cite_note-8)
、[Wikipedia](/wiki/%E3%82%A6%E3%82%A3%E3%82%AD%E3%83%9A%E3%83%87%E3%82%A3%E3%82%A2 "ウィキペディア")
、[GitHub](/wiki/GitHub "GitHub")
などが知られている。大半のオープンソースのLLMは一般公開されているデータを利用しているが、非公開のデータで事前訓練が行われることもある[\[9\]](#cite_note-9)
。事前訓練データは、重複排除、毒性が高いシーケンスの除外、低品質データの破棄など、生テキストをさまざまな手順で前処理して作成される[\[10\]](#cite_note-10)
。言語データの蓄積は年率7%で増加しており、2022年10月現在、高品質な言語データは4兆6,000億語から17兆語の範囲内にあると推定されている[\[11\]](#cite_note-11)
。LLMでは事前訓練データを広範に使用するため、事前訓練データに評価データが混入すると、ベンチマーク評価時のモデル性能に影響を与えるデータ汚染が起こる[\[12\]](#cite_note-12)
。

\]

詳細は「[ニューラルスケーリング則](/wiki/%E3%83%8B%E3%83%A5%E3%83%BC%E3%83%A9%E3%83%AB%E3%82%B9%E3%82%B1%E3%83%BC%E3%83%AA%E3%83%B3%E3%82%B0%E5%89%87 "ニューラルスケーリング則")
」を参照

一般にLLMは、モデルの大きさ、訓練データセットの大きさ、訓練費用、訓練後の性能という4つのパラメータにより特徴づけられる。これらの4つの変数はそれぞれ実数で正確に定義することができ、経験から「スケーリング則（scaling laws）」と呼ばれている単純な[統計的法則](/w/index.php?title=%E7%B5%8C%E9%A8%93%E7%9A%84%E7%B5%B1%E8%A8%88%E5%89%87&action=edit&redlink=1 "「経験的統計則」 (存在しないページ)")
（[英語版](https://en.wikipedia.org/wiki/Empirical_statistical_laws "en:Empirical statistical laws")
）によって関係していることがわかっている。
